# mcpp-mblsha

Fork of http://mcpp.sourceforge.net with minor compatibility fixes.

# Installation

1. Mac: `brew install https://github.com/mblsha/mcpp/raw/master/mcpp-mblsha.rb`
2. Windows: pre-built [Windows Stand-alone binary](https://github.com/mblsha/mcpp/releases/download/v2.7.2-mblsha/mcpp.exe).

Windows building instructions: https://github.com/mblsha/mcpp/blob/master/INSTALL (section 3)
